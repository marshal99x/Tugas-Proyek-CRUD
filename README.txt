CARA MENJALANKAN PROJECT

1. Install XAMPP
2. Jalankan Apache dan MySQL
3. Copy folder mini-crud-buah ke:
   C:\xampp\htdocs\

4. Buka phpMyAdmin:
   http://localhost/phpmyadmin

5. Import file database.sql

6. Jalankan di browser:
   http://localhost/mini-crud-buah

FITUR:
- CRUD Data Buah
- Tambah Data
- Edit Data
- Hapus Data
- Prepared Statement
